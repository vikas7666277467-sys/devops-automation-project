# 🚀 DevOps Automation Project

A complete DevOps Automation Project demonstrating Infrastructure as Code (IaC), Configuration Management, Containerization, CI/CD, and Linux Automation.

---

## 📌 Project Overview

This project showcases modern DevOps practices using industry-standard tools such as Jenkins, Docker, Docker Compose, Terraform, Ansible, GitHub Actions, and Bash Scripting.

The project demonstrates how infrastructure, automation, and deployment can be managed efficiently using DevOps principles.
